# todayHealth_ts [개인프로젝트]
## 기획 배경
  오.운.완 이라는 단어가 생길만큼 운동에 대한 수요는 증가하고 있습니다.
  
  수요가 증가한 만큼, 운동을 사랑하는 사람들끼리의 소통 공간을 만들고 싶었습니다.
  
  게시물 이용 시 매번 운동 후 사진 인증이 필수인만큼, 운동에 진심인 사람들을 위한 커뮤니티 입니다. 
  
  ## 기술 스택
  * React 18.2.0
  * Vite 4.3.2
  * TypeScript 5.0.2
  * Router 6.10.0
  * Firebase 9.22.0
  
  ## Install
  * 패키지 : > yarn
  * 실행 : yarn dev

## Pages

1. 메인페이지
  * 운동 텍스트 기록 가능
  * 스톱워치 기능

2. 회원가입 페이지
  * 아이디 / 패스워드 형식 검사
  * 비밀번호 재확인 기능

3. 게시물 페이지
  * 제목 기반 검색 기능
